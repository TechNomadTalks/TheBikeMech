# The Bike Mech

Professional bicycle repair and maintenance services on the South Coast of KwaZulu-Natal, South Africa.

## Tech Stack

- **Next.js 16** - React framework with App Router
- **TypeScript** - Type-safe JavaScript
- **Tailwind CSS 4** - Utility-first CSS
- **Framer Motion** - Animations
- **Sanity CMS** - Content management
- **Lucide React** - Icons

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Start production server
npm run start
```

## Environment Variables

Copy `.env.example` to `.env` and configure:

- `NEXT_PUBLIC_SANITY_PROJECT_ID` - Sanity project ID
- `NEXT_PUBLIC_SANITY_DATASET` - Sanity dataset (production/dev)
- Sanity API tokens for write access

## Features

- **Services** - Full service menu with pricing
- **Bike Sales** - Listings for pre-owned bikes
- **Gallery** - Photo gallery with categories
- **Blog** - News and tips
- **Events** - Calendar with upcoming cycling events
- **FAQ** - Frequently asked questions
- **About** - Company information and team
- **Contact** - Contact form with email notifications
- **WhatsApp Integration** - Quick chat button
- **Search** - Global search across services, bikes, blog, FAQ
- **Testimonials** - Customer reviews carousel
- **Back to Top** - Smooth scroll button
